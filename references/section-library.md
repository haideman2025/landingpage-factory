# Section Library — đa định dạng cho LP chuyển đổi cao

Mỗi LP do build_lp.py tạo gồm các section sau (đa kiểu trình bày để giữ người xem cuộn & tăng chuyển đổi). Có thể thêm/bớt khi tùy biến.

| # | Section | Vai trò tâm lý | Định dạng |
|---|---------|----------------|-----------|
| 1 | Marquee | Hook ưu đãi luôn hiện | dải chạy |
| 2 | Nav | Thương hiệu + trust | sticky |
| 3 | Hero | H1 outcome + giá + CTA + ảnh | ảnh 4:5 + text |
| 3b| Brand DNA box | (dev) prompt nhất quán | hộp copy |
| 4 | Press strip | Uy tín báo chí | badges |
| 5 | Pain cards | Đồng cảm nỗi đau | 3 card |
| 6 | Agitate quote | Khoét sâu hậu quả | ảnh full-bleed + quote |
| 7 | Solution | Giới thiệu sản phẩm | ảnh product + text |
| 8 | Mechanism/feature | Vì sao hiệu quả | list + pH/feature bar |
| 9 | Variant tabs | Cho chọn (mùi/màu/gói) | tabs tương tác |
| 10| Benefits grid | Quét nhanh lợi ích | 4 icon tile |
| 11| How-to 3 step | Dễ dùng | timeline + 3 ảnh |
| 12| Before/After | Hình dung thay đổi | 2 cột ảnh |
| 13| Stat counters | Social proof số | đếm số chạy |
| 14| Case study | 1 câu chuyện sâu | ảnh + quote + kết quả |
| 15| Testimonial carousel | Nhiều bằng chứng | vuốt ngang |
| 16| Video | Bằng chứng động | ô video |
| 17| UGC grid | Ảnh khách thật | grid 6 |
| 18| Comparison/summary | Hạ đối thủ / chốt lý do | bảng / grid |
| 19| Founder note | Niềm tin thương hiệu | ảnh + lời |
| 20| Value + Gift + Countdown | Tăng AOV + urgency | bảng giá + đếm ngược |
| 21| Order form COD | Chuyển đổi | form ≤5 trường |
| 22| Guarantee | Gỡ rủi ro | risk reversal |
| 23| FAQ | Pre-empt phản đối | accordion |
| 24| Final CTA + Footer | Chốt + pháp lý | CTA + footer |
| — | Sticky bottom CTA | Luôn hiện nút mua | fixed |

**Quy tắc:** ≥6 CTA rải khắp; form đặt sau khi cảm xúc chín (sau social proof + value); mỗi section 1 nhiệm vụ; mobile-first.

**Tùy biến ngành khác:** đổi `angle_type` (BA/CMP/SCI), đổi tabs variant (mùi → màu/gói/size), đổi mechanism (pH → thành phần/công nghệ), đổi copy trong ANGLES. Giữ nguyên khung section.
