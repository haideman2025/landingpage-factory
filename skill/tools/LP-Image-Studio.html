🍌 LP Image Studio

Upload template HTML → tự quét ô ảnh + prompt → tạo toàn bộ ảnh bằng Nano Banana Pro 2 (Gemini 3 Pro Image) với ảnh sản phẩm tham chiếu → tải ZIP (HTML hoàn chỉnh + ảnh chuẩn SEO) deploy ngay.

0 Cấu hình API (AI Studio)

⚠️ Mở tool qua https (deploy lên Cloudflare Pages, hoặc Live Server) để gọi API ổn định. Mở trực tiếp file:// có thể bị chặn CORS. Key nằm 100% trên máy bạn, không gửi đi đâu ngoài Google.

1 Upload template HTML

Kéo-thả hoặc bấm chọn file .html (template do Claude tạo, có ô ảnh + prompt)

2 Upload ảnh sản phẩm (tham chiếu)

Tải 1–5 ảnh chai/sản phẩm ONIIZ (nền sạch, rõ nhãn). AI sẽ giữ đúng bao bì/logo khi tạo ảnh.
Kéo-thả hoặc bấm chọn ảnh sản phẩm (jpg/png)

3 Tạo ảnh

4 Tải ZIP hoàn chỉnh

ZIP gồm: index.html (đã thay ô ảnh bằng <img> chuẩn SEO: alt, loading=lazy) + thư mục assets/ ảnh .jpg. Giải nén → deploy Cloudflare Pages/Netlify.
Mẹo: ảnh nào chưa ưng, sửa prompt trong ô của nó rồi bấm "Tạo lại". Khi đủ ảnh đẹp mới tải ZIP.
